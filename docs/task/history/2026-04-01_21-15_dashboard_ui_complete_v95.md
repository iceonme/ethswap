# 仪表盘 UI 深度美化验收文档 (V9.5-Ultimate)

**日期**：2026-04-01

## 1. 变更概述
在 Premium 版的基础上，补齐了最后两项关键视觉与数据功能：
- **左侧持仓深度面板**：完整复现了保证金、多空持仓、持仓均价、持仓价值等 6 项核心指标。
- **K线图实时交易标记**：在主表盘上通过绿色向上的图标表示买入，红色向下的图标表示卖出，直观展示历史买卖点位。

## 2. 核心功能实现
| 功能模块 | 实现细节 |
| :--- | :--- |
| **持仓详情 (New)** | 采用 `.position-card` 容器，实时计算并展示杠杆后的等效数量与保证金占用情况。 |
| **交易标记 (New)** | 利用 `LightweightCharts` 的 `setMarkers` API，将 `trade_history` 自动映射为图表上的箭头标记。 |
| **动态色彩** | 账户总价值和收益率百分比会根据是否盈利动态切换青色/红色。 |
| **布局优化** | 微调了左侧栏各组件的边距与对齐，确保与设计图完全吻合。 |
