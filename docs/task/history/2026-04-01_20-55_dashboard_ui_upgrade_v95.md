# 仪表盘 UI 全面升级验收文档 (V9.5-Premium)

**日期**：2026-04-01

## 1. 变更概述
按照高保真设计图要求，对仪表盘进行了深度美化与功能增强，从“Minimalist”版本跨越到“Premium”专业版。

## 2. 核心功能实现
| 功能模块 | 实现细节 |
| :--- | :--- |
| **视觉风格** | 采用深蓝色系 (`#0b0f1a`)，引入玻璃拟态 (Glassmorphism) 效果与霓虹光效。 |
| **专家建议** | 新增左侧建议框，实时渲染策略生成的“专家建议”文本及状态色。 |
| **交易统计** | 右侧栏新增统计仪表盘，自动计算胜率、盈亏比、成交数及已实现 PNL。 |
| **三图联动** | 完善 K线、RSI、权益曲线三图布局。RSI 增加 70/30 警戒线，权益曲线支持渐变填充。 |
| **数据对齐** | 优化 Socket 数据包处理，支持历史权益曲线的一键加载。 |

## 3. 验收细项
- [x] **配色对齐**：色彩饱和度与设计图一致，区分极高、极低位彩色标签。
- [x] **字体优化**：关键数值采用 `JetBrains Mono`，文字采用 `Noto Sans SC`。
- [x] **动态更新**：RSI 阈值、重置次数等紧凑信息实时变动。
- [x] **交易卡片**：每条记录显示成交量、成交价、PNL 及交易理由。
